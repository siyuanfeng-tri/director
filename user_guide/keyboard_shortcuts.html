Keyboard Shortcuts¶

This documents the shortcuts available in Director and related applications.

Director¶

  • Ctrl + Q – quits the application
  • Ctrl + O – load a polyData data file (vtp, ply)
  • Ctrl + S – save currently selected object-model polyData as a data file
  • Ctrl + B – opens bot-spy (LCM traffic inspection tool)
  • Ctrl + I – opens signal-scope
  • Ctrl + M – opens the Multisense panel
  • Ctrl + F – opens the footsteps panel
  • F8 – opens the Python console
  • F9 – opens the Matlab console
  • F10 – opens the task panel
  • Footsteps Driver, Playback Panel:
    • Ctrl + Enter – executes the footstep plan
  • Teleop Panel:
    • Ctrl + Shift + S – plan standing posture
    • Ctrl + Shift + N – plan nominal posture
    • Ctrl + Shift + L – plan pregrasp with left hand
    • Ctrl + Shift + R – plan pregrasp with right hand
    • Ctrl + Shift + P – equivalent to pressing the Plan button
  • Python Console:
    • Ctrl + D / Ctrl + W – closes the python console

Console App¶

  • Ctrl + Q – quits the application
  • F1 – shows the object model
  • F8 – opens the Python console

Drake Visualizer¶

  • Ctrl + Q – quits the application
  • F1 – shows the object model
  • F2 – toggles the screen grabber
  • F3 – toggles the camera bookmarks
  • F8 – opens the Python console
